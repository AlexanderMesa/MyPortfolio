About Me

As far back as I can remember, I enjoyed math, science, and computers. I like the challenge of problem-solving and the feeling of accomplishment when getting something to work. I am disciplined, methodical, and determined. My decision to pursue chemical engineering reflects my passions and interests. I enrolled in the UNC Coding Boot Camp to expand my knowledge on programming and to learn more about web development.

The UNC Coding Boot Camp is a six-month full stack web development course. For the back end, I learned how to access APIs from a website and retrieve data from it. I also made my own databases using MySQL and MongoDB with create, read, update, and delete (CRUD) functionality. For the front end, I established a user interface and organized data on webpages using CRUD. I used Express for the server side to establish a connection between the front end and back end. At the end of the course, I utilized React to make complicated websites run smoothly.

My hobbies include running, hiking, playing violin, watching soccer, and going on adventures with friends.
