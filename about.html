About Me

I graduated from North Carolina State University with a bachelor's degree in chemical engineering. I chose that major to incorporate my interests in math, chemistry, and computers. I took several programming classes before college such as Visual Basic, C++, and Java. Problem-solving and troubleshooting were integral aspects of my major. I enrolled in the UNC Coding Boot Camp to expand my knowledge on programming and learn more about web development.

The UNC Coding Boot Camp is a six-month full stack web development course. I spent about three weeks learning about HTML and CSS and the rest of the course learning about JavaScript. I learned front end and back end web development. For the back end, I learned how to access Application Program Interfaces (APIs) from a website and retrieve data from that database. I also made my own databases using MySQL and MongoDB with create, read, update, and delete (CRUD) functionality. For the front end, I organized data on webpages using the CRUD functionality. I used Express.js for the server side to establish the connection between the front end and back end. At the end, I utilized React to make complicated websites run more smoothly.
